# Eldinamo Link Hub

Interactive Linktree-style page with dark neon blue visuals, motion effects, and social links.

## Local Preview

Open `index.html` directly in browser, or run a local server:

```powershell
python -m http.server 5500
```

Then open:

`http://localhost:5500`

## Deploy To Vercel

1. Push this project to GitHub.
2. Go to [Vercel](https://vercel.com/), import the GitHub repository.
3. Framework preset: **Other** (static site).
4. Keep default settings and deploy.
